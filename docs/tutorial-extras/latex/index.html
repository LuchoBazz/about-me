LaTeX Math Formulas

Docusaurus supports LaTeX math equations using KaTeX. You can write both inline and block-level mathematical expressions.

Inline Equations​

Use single dollar signs $...$ for inline math:

  • Einstein's mass-energy equivalence: E=mc2E = mc^2E=mc2
  • Pythagorean theorem: a2+b2=c2a^2 + b^2 = c^2a2+b2=c2
  • Quadratic formula: x=−b±b2−4ac2ax = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}x=2a−b±b2−4ac​​
